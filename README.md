# Route Market

React Router를 이용해 만든 간단한 쇼핑몰 실습 프로젝트입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드 방법

```bash
npm run build
```

## 주요 라우트

- `/`: 홈
- `/products`: 상품 목록
- `/products/:productId`: 상품 상세 페이지
- `/cart`: 장바구니
- `/login`: 로그인
- `/report`: 제작 보고서

## 사용 기술

- React
- React Router
- Vite
